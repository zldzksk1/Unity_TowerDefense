
Tower Defense Game developed with Unity

This is a simple Tower Defense Game. Oppenents will move along the path, 
and users can place their weapone tower against the enemies.
Every time you distroy the enemy, their get slightly stronger to make game more challange.

I have developed this game based on Realm Rush Game lecture of GameDev.TV
You are able to play this game at https://play.unity.com/mg/other/tower_defense-1
